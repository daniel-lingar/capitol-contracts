## Short Company Description

Capitol Contracts LLC develops structured, non-clinical psychoeducational education materials, implementation tools, and training frameworks for organizations serving adults in recovery, reentry, veteran-support, peer-support, housing-stabilization, and community-based support settings.

## One-Sentence Service Description

Capitol Contracts LLC provides non-clinical curriculum development, facilitator-support materials, pilot deployment documentation, and plain-language pattern-recognition education tools for partner organizations.

## Expanded Service Description

Capitol Contracts LLC designs structured educational materials that help organizations teach survival-pattern recognition, nervous-system adaptation, shame-cycle interruption, executive-function disruption concepts, and practical pause/exit tools without requiring participants to disclose personal trauma histories.

The company focuses on facilitator-ready materials, participant handouts, implementation checklists, pilot documentation, and partner-facing support tools that can operate alongside existing clinical, peer-support, recovery, case-management, or community-support services.

## WRH Description

**What Really Happened (WRH)** is a structured, non-clinical psychoeducational curriculum system that teaches pattern recognition, system logic, nervous-system adaptation, executive-function disruption, shame cycles, freeze responses, avoidance loops, and practical interruption points.

WRH uses a non-exposure model. Participants do not have to recount personal trauma histories to learn the material.
