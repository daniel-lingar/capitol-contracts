# Capitol Contracts LLC

Official public documentation hub for Capitol Contracts LLC.

Capitol Contracts LLC develops structured, non-clinical psychoeducational education materials, implementation tools, and training frameworks for public education, peer-support settings, recovery environments, veteran-support programs, reentry settings, and institutional partners.

- non-clinical educational support services
- curriculum development

- plain-language pattern-recognition tools
- implementation support for partner organizations

## Scope Boundary

Capitol Contracts LLC materials are not therapy, not medical advice, not legal advice, and not a replacement for licensed clinical care.

The company develops structured educational tools that help people and organizations recognize survival patterns, reduce shame-based misreadings, and support safer implementation practices.
